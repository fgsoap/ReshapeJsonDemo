# Reshape JSON Demo

This is a demo of how to reshape JSON with Azure Functions.

```mermaid
graph TD;
    PowerApps-->SharePoint;
    PowerApps-->CustomConnector;
    CustomConnector-->AzureFunctions;
    AzureFunctions-->CustomHandler;
```
